# 🛒 買い物リストアプリ - ビジュアルガイド

**シンプルに妻と買い物リストを共有。アレクサで追加、スマホとウォッチで確認・削除。**

---

## 📱 このアプリでできること

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│  📌 買い物リストの中心は「Home Assistant」                   │
│                                                             │
│     リスト内容は HA サーバーで一元管理                       │
│     スマホ・ウォッチ・Alexa が同じデータを見ている           │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 🎯 主な3つの使い方

```
  【1】Alexa で音声追加                【2】スマホ・ウォッチで確認・削除
  ┌──────────────────────┐            ┌──────────────────────┐
  │ "アレクサ、買い物     │            │ 🛒 買い物リスト      │
  │  リストに牛乳を追加" │            │ ─────────────────    │
  │                      │            │ □ 牛乳                │
  │ ↓（自動でHA更新）    │            │ □ パン                │
  │                      │            │ ✓ 削除ボタン          │
  └──────────────────────┘            └──────────────────────┘
        Alexa                        Phone + WearOS Watch
```

## 🏗️ システム全体図

```
                    インターネット
                          │
         ┌────────────────┼────────────────┐
         │                │                │
    🏠 Home              📱 Phone          ⌚ Watch
    Assistant           App              App
    (HA)                                  
    ├─ todo.home       ├─ Settings       ├─ Shopping List
    │  (買い物リスト)   ├─ Widget         └─ Item Delete
    │                  └─ Item Delete
    │
    └─ 🏠 家庭内ネットワーク
       (192.168.1.x)
```

### 🔄 データフロー

```
【追加】
Alexa → Home Assistant → スマホ・ウォッチ に 15分ごとに 反映
        (todo.home エンティティ)

【削除】
スマホ/ウォッチ → Home Assistant (削除命令) → HA が自動処理
                                            ↓
                                        Bring! からも削除
                                        (Alexa でも反映)
```

## 🎯 使用シナリオ

### シーン 1: 日常の買い物

```
朝 08:00          家族
─────────────────────────────
夫がパソコンで    「アレクサ、卵をリストに追加」
作業中             ↓ (声で追加・手が離せない)
                  
妻が買い物に      スマホ・ウォッチで確認
出かける前        ↓ 
                  「卵・牛乳・パン...」
                  
スーパーで        ウォッチで確認しながら買う
                  「あ、卵あった」→ウォッチで削除
                  
帰宅後            リスト空で完了確認
```

### シーン 2: 急な買い足し

```
夫: 「あ、バターなくなった」
    Alexa: 「了解、リストに追加しました」
           ↓
妻のスマホに反映  (15分以内)
    ↓
次の買い物で購入  ✓ 削除

（双方向同期で、誰が追加・削除してもOK）
```

## 📈 技術的なこだわり

| 項目 | 実装 | メリット |
|------|------|---------|
| **Token 管理** | EncryptedSharedPreferences | セキュア・デバイス紛失時も安全 |
| **同期** | WorkManager 15分ポーリング | バッテリー消費最小化 |
| **UI 更新** | Jetpack Compose | シンプル・高速・バグ少ない |
| **ウィジェット** | Glance API | 最新・軽量・反応的 |
| **テスト** | ユニット + Lint + UI テスト | 品質保証・自動化 |
| **CI/CD** | GitHub Actions | コード push で自動ビルド・配布 |

## 🎁 このアプリの特徴（つまり何が良いのか）

| 特徴 | 説明 |
|------|------|
| **声で追加** | スマホ触らずにアレクサで「牛乳を追加」→ OK |
| **見守れる** | スマホ・ウォッチ両対応。妻の買い物進捗を確認可能 |
| **削除が楽** | ウィジェット・ウォッチから1タップ。わざわざアプリ開かない |
| **セットアップが簡単** | Token 自動入力。初心者でも OK |
| **セキュア** | Token 暗号化保存。盗聴・紛失時も安全 |
| **リアルタイム性** | 15分以内に反映。十分実用的 |
| **Bring! と連携** | Alexa で追加したリストがそのまま使える |
